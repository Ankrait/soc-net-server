[Ссылка на GitHub Pages](https://ankrait.github.io/SocialNetwork/)

### Getting Started
Для локального запуска
1. Клонирование репозитория
```sh
git clone https://github.com/Ankrait/MyProject.git
```
2. Установка зависимостей
```sh
yarn install

// or with npm

npm install
```
3. Старт сервера
```sh
yarn start

// or with npm

npm start
```
